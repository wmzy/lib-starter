# lib-starter

> Library starter

## Usage

```bash
PROJECT_NAME=new-project-name git clone --depth=1 https://github/wmzy/lib-starter $PROJECT_NAME \
&& cd $PROJECT_NAME \
&& rm -rf .git \
&& git init \
&& npm i
```

## Workflow

```bash
# develop
npm start

# build
npm run build

# test
npm test

# commit changes
npm run commit

# publish
npm publish
```
